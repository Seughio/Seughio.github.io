# Seughio 的课程笔记

这里整理课程学习过程中形成的笔记、公式和复习资料。内容以课程章节为主线，尽量把概念、公式、图像和典型计算放在同一个脉络下，方便课后查阅和考前复习。

目前主要整理的是专业必修课程中的《工程热力学（甲）》。这门课涉及热力学基本定律、工质热力性质、理想气体与蒸气过程、气体与蒸气流动、压缩机、动力循环、制冷循环以及化学反应系统热力学等内容。网页笔记适合按章节快速查找，PDF 笔记适合集中阅读和打印复习。
