# 反馈与纠错

如果你在阅读时发现公式、符号、图示、文字表述或页面排版存在问题，可以通过 GitHub Issues 提交反馈。

[提交笔记反馈](https://github.com/Seughio/Seughio.github.io/issues/new?template=note-correction.md){ .md-button .md-button--primary }

提交时请尽量说明：

- 所属课程和章节；
- 问题所在的小节、公式或图片；
- 当前内容存在的问题；
- 建议的修改方式或参考依据。

如果问题涉及公式，建议直接写出正确公式；如果涉及排版或图片，可以附上截图。反馈内容确认后会在后续更新中修正。

也可以先在[已有反馈](https://github.com/Seughio/Seughio.github.io/issues)中搜索，避免重复提交。
