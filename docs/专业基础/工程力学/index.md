# 工程力学

工程力学说白了，其实就是理论力学里面的静力学内容加上材料力学的部分内容糅合起来的，如果觉得工程力学的历年试题比较少，可以去找找理力和材力的题目（我记得当时复习期末的时候看了一下材料力学的历年卷就刚好押中了一个大题）。

wcj老师特别好的老师，上课讲的比较清楚，也很和善，经常与他交流的话你的名字就可以被他记住（不过可能上课会被叫起来回答问题hhh）。

这门课除了线下教学和作业之外，还有MOOC上的讨论题和作业，每周都会有，而且讨论不一定是每周只有一次，需要注意自己的班级的日期（我就不小心漏了四个讨论，老师期末的时候主动在钉钉上私信我要我补起来，虽然丢了点分但是感谢老师提醒）。
