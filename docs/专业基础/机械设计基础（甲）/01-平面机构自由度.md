# 第 1 章 平面机构自由度

## 1.1 平面机构自由度计算

平面机构中，每个活动构件在平面内具有 3 个自由度。因此，若机构中有 $n$ 个活动构件，则构件总自由度为 $3n$ 。

运动副会引入约束：

- 低副约束数：每个低副引入 2 个约束，记低副数为 $P_L$，约束数为 $2P_L$
- 高副约束数：每个高副引入 1 个约束，记高副数为 $P_H$，约束数为 $P_H$

因此，平面机构自由度为：

$$
F=3n-(2P_L+P_H)
$$

## 1.2 局部自由度

构件局部运动所产生的自由度，不影响机构的输入输出关系。在计算机构自由度时，应识别并排除局部自由度对计算结果的影响。

## 1.3 复合铰链

两个或两个以上构件在同一轴线上用回转副相联接构成。若有 $m$ 个构件构成同轴复合铰链，则应计为 $m-1$ 个回转副。

## 1.4 虚约束

某些约束对机构运动与其他约束重复。计算自由度时，应注意识别虚约束，避免重复计入约束数。

常见情形：

- 两构件之间的约束功能与其他约束重复
- 对机构实际运动不起独立限制作用的约束

## 1.5 计算步骤

1. 确定活动构件数 $n$
2. 统计低副数 $P_L$
3. 统计高副数 $P_H$
4. 识别局部自由度、复合铰链和虚约束
5. 代入公式 $F=3n-(2P_L+P_H)$
