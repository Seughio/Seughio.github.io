# 普通化学（乙）

《普通化学（乙）》短学期课程，内容大部分与高中化学重合，在此基础上进行拓展延申，如果高中化学底子比较好的话，学起来会比较轻松。

lg老师班，有 6 次小测（几乎是一周一次了），选其中最好的 4 次作为小测成绩。小测与历年小测题高度重合，可以提前看一看，算一算，记一记。

最后期末复习可以面向历年卷复习，整体难度不算大，但是对一些化学原理需要有自己的理解。
