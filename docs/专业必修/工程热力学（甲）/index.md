# 工程热力学（甲）

《工程热力学（甲）》作为能源工程学院三大专业课程之一，其难度不容小觑。特别是在概念理解、查表计算、循环分析等方面，知识点比较零碎和混杂，需要记忆的知识点很多。学习时可以把“状态参数关系、能量方程、熵变分析、过程图像和循环效率”作为主线。各章公式主要服务于工程计算，重点是判断过程类型、选取合适的热力学模型，并把热量、功量、内能、焓和熵之间的关系统一起来。
