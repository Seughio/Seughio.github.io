# 工程流体力学（甲）

《工程流体力学（甲）》分为 I 和 II ，分秋冬两个学期上。I 主要是前六章的内容，II 主要是后三章的内容。yzt老师上课主要讲的是重点知识点，上课听了的话就大概率可以摸清考试要考的重点内容。I 会有两次小测，难度都不会很大，主要是考一些概念性的知识点；II 会有一次小测，可能涉及到计算题，需要提前拟合一下课后习题。工流学的内容感觉都是挺难的，毕竟流体力学目前也还是科学界的一大难题，不过考试的内容并不算很复杂，需要记忆的公式大多数都是简化版的。
